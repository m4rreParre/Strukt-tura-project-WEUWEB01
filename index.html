Vad är struktura ?

Struktura är din guide till att bemästra webblayouts! Här får du lära dig hur du bygger responsiva och moderna webbsidor med hjälp av de mest kraftfulla verktygen inom CSS. Oavsett om du är nybörjare eller har erfarenhet, kommer du hitta resurser som hjälper dig att förbättra dina färdigheter.

Vad kommer du att lära dig?

➤ Flexbox

Utforska CSS Grid och hur du bygger komplexa, responsiva layouter med hjälp av ett rutnätssystem.

➤ CSS Grid

Lär dig att använda Flexbox för att skapa flexibla och dynamiska layouter. Perfekt för allt från enkla centrerade element till avancerade sidstrukturer.

➤ Responsiv design

Förstå grunderna i responsiv design och lär dig hur du anpassar dina webbsidor för olika skärmstorlekar med hjälp av media queries

Interaktiva övningar

Vi tror på lärande genom att göra! Testa dina kunskaper i vår sandbox där du kan experimentera fritt med HTML och CSS, eller anta våra utmaningar som hjälper dig att bygga riktiga webblayouts.

Verktyg & Resurser

Få tillgång till användbara verktyg som hjälper dig på vägen – från cheat sheets för Flexbox och CSS Grid till interaktiva testverktyg där du kan se din layout anpassa sig i realtid.
