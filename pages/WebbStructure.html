Hur fungerar Flexbox?

Flexbox (Flexible Box Layout) är ett CSS-layoutsystem som används för att skapa flexibla och responsiva layouter. Det används främst för att ordna element i en rad (row) eller kolumn (column) på ett smidigt sätt.

Grundläggande begrepp inom Flexbox:

  • Flex container: Det element som innehåller flexibla barn. Man aktiverar Flexbox genom att sätta display: flex; på containern.

  • Flex items: De element som ligger inuti flex-containern och blir flexibla.

  • Main axis: Den huvudsakliga axeln i flex-containern, antingen rad eller kolumn.

  • Cross axis: Den sekundära axeln i flex-containern, antingen rad eller kolumn.

  • display: flex; på .header gör att den blir en flex-container.

  • align-items: center; centrerar innehållet vertikalt i headern.

  • .nav är också en flex-container för att länkarna ska hamna på en rak linje med display: flex;.

Vad är CSS Grid?

CSS Grid Layout är ett kraftfullt tvådimensionellt layoutsystem som används för att skapa komplexa layouter på webbsidor. Det gör det möjligt att skapa både rader och kolumner och placera element exakt.

För att komma igång med CSS Grid börjar du med att göra en container till en grid genom att använda display: grid;. Sedan kan du definiera rader och kolumner med till exempel grid-template-columns och grid-template-rows.
