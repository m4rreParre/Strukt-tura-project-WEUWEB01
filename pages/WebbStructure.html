Hur fungerar Flexbox?

Flexbox (Flexible Box Layout) är ett CSS-layoutsystem som används för att skapa flexibla och responsiva layouter. Det används främst för att ordna element i en rad (row) eller kolumn (column) på ett smidigt sätt.

Grundläggande begrepp inom Flexbox:

  • Flex container: Det element som innehåller flexibla barn. Man aktiverar Flexbox genom att sätta display: flex; på containern.

  • Flex items: De element som ligger inuti flex-containern och blir flexibla.

  • Main axis: Den huvudsakliga axeln i flex-containern, antingen rad eller kolumn.

  • Cross axis: Den sekundära axeln i flex-containern, antingen rad eller kolumn.

Vad gör denna kod?

  • display: flex; på .header gör att den blir en flex-container.

  • align-items: center; centrerar innehållet vertikalt i headern.

Vad är CSS Grid?

CSS Grid Layout är ett kraftfullt tvådimensionellt layoutsystem som används för att skapa komplexa layouter på webbsidor. Det gör det möjligt att skapa både rader och kolumner och placera element exaktp

Grundläggande begrepp inom CSS Grid:

  • Grid container: Det element som innehåller en grid-layout. Man aktiverar Grid genom att sätta display: grid; på containern.

  • Grid items: De element som ligger inuti en grid-container och anpassar sig efter rutnätet.

  • Grid lines: De osynliga linjer som delar upp rader och kolumner.

  • Grid tracks: De faktiska raderna och kolumnerna i rutnätet.

  • Grid areas: Områden i rutnätet som kan innehålla flera grid items.

Exempel på en header med CSS Grid

En header kan skapas med Grid för att placera logotypen, navigeringen och eventuellt en sökruta i ett organiserat rutnät.

Vad gör denna kod?

  • display: grid; på .header gör att den blir en grid-container.

  • grid-template-columns: 1fr 2fr 1fr; skapar tre kolumner med olika bredder.

  • grid-template-rows: 100px; skapar en rad med en fast höjd.

  • grid-column: 2; placerar .nav i den andra kolumnen.
